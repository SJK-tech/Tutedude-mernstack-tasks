# Tute Task 5

A simple HTML and css web application for a laundry service that streamlines user experience.

Consists of navbar and hero section with description and services. Implemented flexbox
Used @media property to change webpage based on resolution of screens. Compatible with tablets and phones.
Includes a hamburger icon which is visible on mobile mode only, consisting of list functions.

1. Download or clone the repository.
2. Open the `tute task7` folder.
3. Open `index.html` in any modern web browser.
