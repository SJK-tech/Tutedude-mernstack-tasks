# Laundry Wallah / Laundry Mart

Laundry Wallah/ Laundry Mart is a web service based on HTML, providing laundry service. The web page focuses on providing a user friendly interface to book laundry services for the user.

## Features
- About section
- Imported an image from internet
- Unordered list of services
- Table of price list
- Form to input details, submit button and contact details

Run it directly using VS Code or any text editor.
